# Filestack
  
Módulo para trabalhar com Filestrack  

## Como instalar este módulo
  
__Baixe__ e __instale__ o conteúdo na pasta 'modules' no caminho do Rocketbot  



## Overview


1. Enviar Imagem  
Enviar uma imagem

2. Redimensionar Imagem  
Redimensionar uma imagem

3. Cortar Imagem  
Cortar uma imagem

4. Baixar Imagem  
Baixar uma imagem
